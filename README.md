# README

## 概述

该项目是一个缓存项目，主要实现了以下功能：

- 缓存数据的读写
- 缓存数据的过期处理
- 缓存数据的LRU淘汰

## 使用方法

1. 下载代码

   ```shell
   git clone https://github.com/lmmmmm000/cache-project.git
   ```

2. 进入项目目录

   ```shell
   cd cache-project
   ```

3. 运行main.go

   ```shell
   go run main.go
   ```
